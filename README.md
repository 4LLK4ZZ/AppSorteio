# 🎲 App Sorteio

Aplicativo desenvolvido em **Flutter** utilizando **Dart**, criado para realizar sorteios de forma rápida, prática e intuitiva.

O projeto tem como objetivo demonstrar conceitos de desenvolvimento mobile multiplataforma, gerenciamento de estado, interfaces responsivas e geração aleatória de resultados.

---

# 📱 Sobre o Projeto

O **App Sorteio** permite gerar números aleatórios para diferentes tipos de sorteios, sendo útil para:

* Sorteios simples;
* Rifas;
* Brincadeiras e jogos;
* Escolhas aleatórias;
* Demonstrações de geração de números randômicos.

O aplicativo foi desenvolvido utilizando uma única base de código para Android, iOS, Windows, Linux, macOS e Web através do Flutter.

# 🚀 Como Executar o Projeto

## 1. Clone o repositório

```bash
git clone https://github.com/SEU-USUARIO/app-sorteio.git
cd app-sorteio
```

## 2. Instale as dependências

```bash
flutter pub get
```

## 3. Execute a aplicação

```bash
flutter run
```
